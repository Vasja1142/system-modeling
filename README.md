# 🚗 Gas Station Simulation / Моделирование АЗС

## 📌 Описание проекта

Проект представляет собой **имитационную модель автозаправочной станции** с двумя топливными колонками и очередью из двух мест. Модель реализована на Python и использует метод статистических испытаний (Монте-Карло).

## 🔧 Параметры системы

- **λ (Lambda)** = 9 авто/час — интенсивность входящего потока
- **μ₁ (Mu1)** = 6 авто/час — производительность 1-й колонки
- **μ₂ (Mu2)** = 3 авто/час — производительность 2-й колонки
- **Время моделирования** = 2*Nкрит.
- **Очередь** = 2 места, 1 место, 0 мест

### 📊 Описание параметров


| № | Параметр (CSV field) | Ед. измерения | Описание                                                               |
|---|----------------------|---------------|------------------------------------------------------------------------|
| 1 | **a**                | car/hour | Пропускная способность системы (количество обслуженных автомобилей в час) |
| 2 | **p_service**        | probability | Вероятность обслуживания (доля автомобилей, получивших обслуживание)   |
| 3 | **p_reject**         | probability | Вероятность отказа (доля автомобилей, получивших отказ)                |
| 4 | **p_busy_ko1**       | probability | Вероятность занятости одной колонки (доля времени, когда одна колонка занята) |
| 5 | **p_busy_ko2**       | probability | Вероятность занятости двух колонок (доля времени, когда обе колонки заняты) |
| 6 | **n_columns_avg**    | cars | Среднее количество занятых колонок                                     |
| 7 | **p_idle_ko1**       | probability | Вероятность простоя первой колонки (доля времени, когда КО1 свободна)         |
| 8 | **p_idle_ko2**       | probability | Вероятность простоя второй колонки (доля времени, когда КО2 свободна)  |
| 9 | **r_avg**            | cars | Среднее количество заявок в очереди                                    |
| 10 | **t_wait_avg**       | hour | Среднее время ожидания в очереди                                       |
| 11 | **t_service_avg**    | hour | Среднее время обслуживания на колонке                                  |
| 12 | **t_system_avg**     | hour | Среднее время заявки в системе (ожидание + обслуживание)               |
| 13 | **n_avg**            | cars | Среднее количество заявок в системе                                    |
